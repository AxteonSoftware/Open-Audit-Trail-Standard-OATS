# OATS Compliance Checklist

## Minimum Compliance

- [ ] Timestamp is recorded
- [ ] User is recorded
- [ ] Operation is recorded

---

## Recommended

- [ ] Unique event ID present
- [ ] Object is identified
- [ ] Field-level changes captured

---

## Full Compliance

- [ ] Old and new values recorded
- [ ] Reason for change captured
- [ ] Source system recorded

---

## Data Quality

- [ ] Consistent timestamp format (ISO 8601)
- [ ] No missing critical fields
- [ ] No data truncation

---

## Interoperability

- [ ] Export to canonical format supported
- [ ] Field mapping documented
